CACAOTRACE TUMACO - EJECUCIÓN EN WINDOWS
=========================================

1. Descomprime la carpeta del proyecto.

2. Abre la carpeta en VS Code o en el Explorador de Windows.

3. Ejecuta:
   1_INSTALAR_TODO.bat

4. Luego ejecuta:
   2_INICIAR_APP.bat

5. Entra al sistema con:
   Usuario: admin
   Contraseña: admin123

6. Dirección local:
   http://127.0.0.1:5000

ORDEN RECOMENDADO PARA PROBAR EL SISTEMA
----------------------------------------
1. Crear productor.
2. Crear finca asociada al productor.
3. Registrar recepción de cacao.
4. Revisar el lote generado automáticamente.
5. Registrar calidad.
6. Registrar poscosecha.
7. Registrar movimiento de inventario.
8. Consultar reportes.

NOTA
----
El proyecto está hecho con Python y Flask. Los archivos HTML y CSS son solo la parte visual necesaria para el diseño de la aplicación web.
